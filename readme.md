### **期中考试增加了文法定义，支持char类型定义和char类型返回值。**

#### 1. 增加类别码：

| 单词名             | 类别码      |
| ------------------ | ----------- |
| `char`             | CHARTK      |
| `NormalChar`单个字符 | CHAR        |
| `'`                | SINGLEQUATE |

#### 2. 文法新增：

```c++
基本类型    BType → 'int' | 'char'
函数类型    FuncType → 'void' | 'int' | 'char'
基本表达式  PrimaryExp → '(' Exp ')' | LVal | Number | ConstChar
常量字符    ConstChar → '''NormalChar'''
```

```c++
const char a = 'a';
char b = 'b';
char c['a'];
char func(char a, char c){
    return 'a';
}

    if ( 'a' < 'b') {
        printf("%c < %c\n", 'a', 'b');
    }
    'a';
    func('a', 'b');
    b = 'c';
    return 0;
}
```
